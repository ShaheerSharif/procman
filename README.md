## Install project

* **Using git**  
`git clone https://github.com/ShaheerSharif/procman.git`

## Install required packages

```bash
# cd into project
# Create venv
python -m venv venv

# Activate venv
source venv/bin/activate

# Install required packages
pip install -r requirements.txt
```

## Run module

```bash
# Start process manager
python -m procman
```
